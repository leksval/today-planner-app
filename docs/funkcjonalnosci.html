Funkcjonalności

Aplikacja Todayplanner oferuje szereg przydatnych funkcjonalności pozwalających na lepsze zaplanowanie kolejnych dni uwzględniając istotne czynniki takie jak pogoda w danym dniu, jakość powietrza. Aplikacja pozwala utworzyć profil użytkownika następnie osoba korzystająca może dodać aktywność w wybranym przez siebie dniu w kalendarzu. Użytkownik otrzyma pełne dane dotyczące pogody i smogu w jego obecnej lokalizacji. Istnieje również możliwość wysyłania powiadomień w formie sms jeśli użytkownik chciałby poinformować inne osoby o zbliżającej się aktywności zaplanowanej przez niego. Poniżej znajdują się wszystkie funkcjonalności aplikacji w bardziej przejrzystej formie.

Funkcjonalności :


- Logowanie
- Pobieranie lokalizacji
- dodawanie zajęć i aktywności do wykonania w danym dniu
- Pobieranie z API smogu
- Pobieranie z API pogody
- Powiadomienia sms
